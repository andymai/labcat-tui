import '@labcat/tui';

A reader installs `@labcat/tui`, drops a single `<Session>` into their app, and gets a working Claude Code-style TUI: warm-dark background, coral accent, `●` tool-call cards, `›` prompt, status line.

## v0.1 status

This is the skeleton release. One component is wired end-to-end:

<tui-tool-call tool="Read" args="posts/">
  <ul>
    <li>proxmox-tuning</li>
    <li>k3s-on-pi</li>
  </ul>
</tui-tool-call>

See [Tool call](/components/tool-call/) for the full contract.

## Install

```bash
pnpm add @labcat/tui
```

```ts
import '@labcat/tui';
import '@labcat/tui/styles.css';
```
